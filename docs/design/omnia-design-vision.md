# Omnia Design Vision

## References
Primary:
- llama.cpp UI
Secondary:
- ChatGPT Mobile

## Product Feeling
The app should feel:
- Calm
- Fast
- Lightweight
- Modern
- Focused

The app should not feel:
- Enterprise
- Dashboard-like
- Developer-centric
- Visually noisy

## Core Principles
1. Conversation First
2. Content Over Controls
3. Progressive Disclosure
4. Mobile First
5. Whitespace Driven
6. Low Visual Noise

## Visual Direction
Background: White
Primary Text: Near Black
Assistant Responses: Rich dark text on white
User Messages: Subtle neutral surface

## Success Metric
A user should be chatting with a local model in under 2 minutes.
