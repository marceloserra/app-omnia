# Omnia Product Roadmap

## Vision
One app. Every model.

Omnia is a mobile-first universal AI client that provides a single, consistent experience across local and cloud AI providers.

## Release 0.1 — Design Foundation
Goal: Define Omnia's visual identity and UX language.
Deliverables:
- Design System
- Storybook
- Theme System
- Navigation Strategy
- Design ADRs

## Release 0.2 — Platform Foundation
Goal: Establish domain contracts before implementation.
Deliverables:
- Domain Models
- Provider Contracts
- Repository Contracts
- State Architecture

## Release 0.3 — First Conversation
Goal: Connect to LM Studio and stream responses.
Deliverables:
- Endpoint configuration
- Connection validation
- Streaming chat
- Chat screen

## Release 0.4 — Conversation Experience
Goal: Persist and manage conversations.
Deliverables:
- SQLite
- History
- Rename
- Delete
- Search

## Release 0.5 — Local AI Experience
Goal: Become the best local AI client.
Deliverables:
- LM Studio
- Ollama
- llama.cpp

## Release 0.6 — Multi Provider
Goal: Become a universal AI client.
Deliverables:
- OpenAI
- Anthropic
- Gemini
- OpenAI-compatible endpoints

## Release 0.7 — Model Management
Goal: Manage and switch models.

## Release 0.8 — Production UX
Goal: Production-grade experience.

## Release 0.9 — Attachments
Goal: Images and PDFs.

## Release 1.0
Stable Universal AI Client.
